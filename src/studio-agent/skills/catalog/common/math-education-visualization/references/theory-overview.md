# 理论总纲

## 这套理论在管什么

这套理论服务的不是一般意义上的视觉设计，而是初中和高中数学教学中的可视化表达。

它最关心的不是画面是否华丽，而是学生是否能够：

- 更快抓住重点
- 更少受无关信息干扰
- 更容易把图、式、逻辑对应起来
- 更容易看懂结构、过程和变化

因此，它首先是一套教学导向的可视化原则，其次才是风格规范。

## 适用范围

优先用于：

- 数学板书设计
- 数学课件中的图示设计
- 讲义插图
- 数学科普中的解释型图像
- 数学动画与分镜

默认面对的是仍在建立数学直觉的学习者，所以特别强调：

- 重点明确
- 负担可控
- 对应清楚
- 过程可跟随

## 不要把三篇理论看成同一层分类

这套理论里有三条不同轴线，它们可以叠加，不是互斥关系。

### 1. 呈现轴

回答：内容如何在空间与时间中展开？

- 静态呈现
- 伪动态呈现
- 动态呈现

### 2. 信息轴

回答：当前要表达的信息本质是什么？

- 定性信息
- 顺序信息
- 定量信息

### 3. 配色轴

回答：颜色此刻承担什么任务？

- 展示场景
- 突出场景
- 动态演变场景

例如，一个方案可以同时是：

- 伪动态呈现
- 顺序信息为主
- 动态演变配色

这三者并不冲突，因为它们回答的是不同问题。

## 默认使用顺序

实际设计时，建议严格按下面顺序：

1. 先确认教学任务
2. 再决定呈现模式
3. 再决定画面元素和视觉变量
4. 最后决定颜色怎样介入

不要从颜色开始，也不要从“好不好看”开始。

## 一句话压缩

整套理论可以压缩成一句话：

尽量把学生的注意力留给数学本身，而不是浪费在认图、找重点和排除干扰上。
